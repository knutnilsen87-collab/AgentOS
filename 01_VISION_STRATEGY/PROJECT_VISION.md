# Project Vision

## Working title
AgentOS

## One-sentence vision
A desktop-first agentic project operating system that can understand, plan, execute, verify, review, and improve technical and semi-technical work across projects with human-controlled autonomy.

## Why this should exist
Most project work is fragmented across chat tools, IDEs, terminals, docs, tickets, and memory in people’s heads.
This project aims to unify those layers into one operational system.

## Core promise
The system should let a user delegate meaningful work without losing:
- control
- traceability
- continuity
- safety
- context

## Strategic differentiation
This project should not try to win only on raw model quality.
It should win on:
- orchestration
- project continuity
- persistent memory
- vertical/domain packs
- sandboxed execution
- evaluators and adversarial review
- transparent, replayable runs
