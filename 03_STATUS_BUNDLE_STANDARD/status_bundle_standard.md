# Status Bundle Standard

## Human-readable section
Use KEY=VALUE lines.

## Machine-readable section
Use a JSON block fenced by:
- MACHINE_READABLE_JSON_BEGIN
- MACHINE_READABLE_JSON_END

## Required keys
PROJECT_NAME
PROJECT_ROOT
SOURCE_OF_TRUTH
WORKFLOW_MODE
STATUS_BUNDLE_IS_PRIMARY_DOCUMENT
LAST_UPDATED
CURRENT_PHASE
LAST_ACTION
LAST_RESULT
NEXT_ACTION
BUILD_STATUS
COMPILE_STATUS
RUNTIME_STATUS
DB_STATUS
AUTH_STATUS
TEST_STATUS
CI_STATUS
RISKS
NOTES
