# New Developer Handoff Standard

A new developer should start in this order:
1. Read `status_bundle.txt`
2. Read `README_MASTER.md`
3. Identify the current phase and next action
4. Verify project root and source of truth
5. Run the next approved PowerShell block
6. Paste back updated `status_bundle.txt`

Never assume the project is compile-ready unless `status_bundle.txt` says so.
