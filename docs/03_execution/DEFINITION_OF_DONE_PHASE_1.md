# Definition of done — phase 1

Phase 1 is done when all of the following are true:

- the workspace installs locally on Windows
- the workspace typechecks cleanly
- the desktop shell launches
- a local project can be selected or pointed to
- the scanner can summarize the selected project
- a task can be created in the UI
- the task engine can classify risk and produce a plan
- the task can be saved to local storage
- the current approval mode is visible in the UI
- status bundle and handoff notes reflect the working state
- the result is demoable without explaining hidden steps
